# Анализ продаж бразильского e-commerce (Olist)

## О проекте

Проанализированы данные о заказах интернет-магазина Olist за период сентябрь 2016 - август 2018. Выборка содержит более 100 000 доставленных заказов.

**Цель:** выявить ключевые категории товаров, сезонность продаж и средний чек, сформулировать рекомендации для бизнеса.

## Инструменты

- SQL (PostgreSQL) - извлечение и агрегация данных
- Excel - визуализация и расчеты
- Word / PDF - оформление отчета

## Ключевые выводы

1. **26% всей выручки** приносят три категории: `health_beauty`, `watches_gifts`, `bed_bath_table`
2. Средний чек стабилен (около 161 BRL) с пиками весной и осенью
3. Категория `watches_gifts` — аномалия: высокая цена (199 BRL) при высоких продажах (5859 шт.)
4. Рекомендации:
   - «народным» категориям — акции для увеличения оборота
   - «люксовым» — точечный маркетинг на платежеспособную аудиторию

## Структура репозитория

- `sql/` - все запросы, использованные в анализе
- `reports/` - финальный отчет в PDF

## Как воспроизвести

1. Скачать датасет Olist с Kaggle
2. Загрузить CSV в PostgreSQL
3. Выполнить запросы из папки `sql/` по порядку
4. Построить графики в Excel (см. отчет)
